How to create a Python program that counts the number of characters in string

We will learn to count the occurrences of a character in string in Python. This tutorial will teach you how to use Python to create a program that counts the occurrences of each character in a string. We will identify the distinct characters, distinct characters, unique characters and count the string's words.

What is the count of occurrences of a character in a string?

  • Take a string (example: "Python Programming") and create an empty dictionary.
  • Use the strings characters as keys, and their count as values.
  • If a character is in the string as a key, increment it. Other options include adding the new character as a key to the dictionary and assigning its count value as 1.

Now, we have a python string dictionary. Let's find out what we can do with this information.

  • Duplicate characters from the String
  • Different characters in the string
  • Unique characters for the String
  • The string's number of words.

Example:

Algorithm for counting the occurrences of a character within a string

Enter a string and the character that you wish to search.

  • Set a count variable to 0.
  • Iterate by using a for loop and check with an if statement if the current and searching characters are equal.
  • When you get the matching character, increment the count variable.
  • Print count variable.

Python program to count the number of characters in a string.


string = input("Please enter the String : ")
char = input("Please enter the Character you want to count: ")
count = 0
for i in range(len(string)):
    if(string[i] == char):
        count = count + 1
print("The total Number of Times ", char, " has Occurred = " , count)
